TimeHRTrendZone
01 / FILTER
Bandpass 5–15 Hz
Butterworth 3rd order. Removes baseline wander (<0.5 Hz) and muscle noise (>40 Hz). Zero-phase via filtfilt so no time shift.
02 / EDGE
Derivative
First difference highlights steep QRS slopes. P and T waves (slow) are suppressed. Array length kept with np.append.
03 / BOOST
Square
x² makes all values positive and non-linearly boosts large QRS peaks vs smaller P/T waves. Simple but very effective.
04 / SMOOTH
MWI 150 ms
Moving window integration over 54 samples (360 Hz × 0.15 s). Converts spiky signal into one clean hump per heartbeat.
05 / DETECT
Peak → HR
90th percentile threshold + 330 ms refractory (max ~180 BPM). HR = mean(60000/RR) per 5-min window, valid RR: 333–1500 ms.
